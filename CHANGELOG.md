# Changelog

All notable changes to the Concierge Platform are documented in this file.

The format is based on the principles of **Keep a Changelog**, adapted for an internal enterprise platform.

---

# [1.2.0-rc1] - 2026-07

## Overview

First Release Candidate of the Concierge Platform user interface.

This release establishes the shared design language, reusable front-end framework, and first implementation (**VUIT Walk-Up Assistance**).

---

---

## Changed

### Project Scope

The project evolved from a single walk-up form into the foundation of the Concierge Platform.

The platform is now intended to support multiple Enterprise Service Management implementations across campus.

---

## Design

Adopted the following design principles:

- Hero outside the card
- Touch-first interaction
- Portrait-first layout
- Gold for actions
- Sage for success
- Shared design system

---

## Repository

Repository structure reorganised to support:

- Shared UI framework
- Multiple implementations
- Future automation components
- Platform documentation
